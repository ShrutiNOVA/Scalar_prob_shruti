# Code Review AI

Analyze your code like an AI and get instant feedback with suggestions & scoring.

## Features
- Detects unused variables
- Finds repeated computations
- Identifies nested loops
- Detects magic numbers
- Provides suggestions
- Gives code quality score (0–10)

## Demo
Paste your code and click submit to see analysis instantly.
